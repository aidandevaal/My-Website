-Microbit Image- First off, the variables: millig = "milligrams" but I couldn't use "mg" because it's already a variable in the acceleration function and it had to be separated in the code. angle = the angle that the micro:bit is tilted to. Now, the millig variable must start equal to the acceleration (which is equal to "mg(x)"). Once that has been set, the program continously runs the middle function, which states that the servo points to the opposite angle of the way the micro:bit is being tilted, so that it goes in the direction the user wants it to (it is NOT inverted). The first "High" and "Low" variables are the minimum and maximum tilt ranges that the micro:bit has. The second "High" and "Low" variables and the angles that the servos rotate through. The final string at the bottom is, in a way, reminding the servo to turn to whatever angle the micro:bit is tilted to, however the second and final functions pretty much run simultaneously. I learned how to use variables to my advantage. The best example would be the "millig" variable and the "mg" variable, I needed a variable that meant "milligrams," but wasn't the same as "mg," so I came up with the "millig" variable and it worked out in the end. --Thanks for visiting my page!--
